Logic

I used 3 images for the boxes, the open box contains 2 parts top and bottom, next the closed box. I have used 2 images for kick 2 image for kick, one i the jumping image and next one is the smiling image.

Using the css Z-index i mate that effect that character is into the box and hiding in the box. And the logic is very simple. I have taken a random number first which is the answer, and placed the kick in it then next an array of [1,2,3] and for every 500ms i have called the shuffle function

shuffle = function(o){
    for(var j, x, i = o.length; i; j = parseInt(Math.random() * i), x = o[--i], o[i] = o[j], o[j] = x);
    return o;
  };
