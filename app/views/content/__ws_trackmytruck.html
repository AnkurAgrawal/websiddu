Summary

I developed a low cost vehicle tracking mechanism using Google Maps and GPS device on the truck. This is application is developed in order to help the small Truck owners who cannot effort a costly tracking systems.

How this works ?

A GPS tracking device must be attached to the vehicle and an SIM card of any network with GPS enabled. This device pings the nearest tower with the exact Latitude and Longitude along with the IMEA number of the device number, tower will send it to the server connected with internet next the server stores and maps the data with already store IMEA recored. Later that client can login to their account and monitor all the vehicles that are registered. Below is a diagram showing how this vehicle tracking works.

Hardware

In the present market there are plethora of options to find out a low cost GPS's tracking device (It costs around ₹1500 to ₹2000 in year 2011). We can get any of the device that have a GSM option and able to ping server. Here is an image of how a tracking device looks like. One have to setup other hardware to enable options to alert on low fuel, low battery etc...

The main idea of the project is to track the vehicle using the GPS device attached to the moving object. We need to place a GPS Hardware device in the vehicle that we are going to track, each GPS device has it machine number and it will communicate the server socket using a GPS protocol which is given by the hardware device.

Initial Prototypes

As the idea of tracking clear. I started working on the initial prototypes, marking all the features that present in the application. Some initial thoughts here..

Features available
  • Customer Can Register any Number of vehicles into their account.
  • Each vehicle must have a unique number.
  • Live tracking (locates the position of the vehicle at present)
  • Gives the Status like stopped, moving and also the Speed at which it is going on.
  • Different types of alerts like Over Speed, Fuel Low, Battery Low etc..
  • Route history of the vehicle(the route at which the vehicle has crossed in the given time)
  • Other features like Fleet Usage, Sensor, Trip, Vehicle distance etc..
